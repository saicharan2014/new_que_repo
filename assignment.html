            4.what is d/f b/w xml and html?
            a.HTML (HyperText Markup Language):
             Used to create and design web pages.
             Focuses on the presentation of data.
             Tags are predefined and fixed.
             More lenient in syntax rules (e.g., self-closing tags are not always required).

            XML (eXtensible Markup Language):
            
            Designed to store and transport data.
            Focuses on the structure and transport of data.
            Tags are not predefined and can be created as needed.
            Strict syntax rules (e.g., tags must be properly closed).
